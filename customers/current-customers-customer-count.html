Nombre de clients

Suivez votre nombre de clients au fil du temps. Ce graphique montre l'effet net des conversions et des annulations sur le nombre total d'abonnés pour la période actuelle.

Actions

Vous pouvez sélectionner l'une des actions suivantes pour modifier les données présentées:

  • Cliquez pour télécharger les données de modification des clients sous forme de fichier CSV ou d'image PNG.
  • Cliquez pour filtrer les données des clients par:
    • Tous les clients
    • Plan
    • Chaîne
    • Appareil

    • Pays

  • Sélectionnez une plage de dates pour vos données. Notez que la plage de dates doit être de 5 jours ou plus.
  • Sélectionnez la granularité de l'axe des x sur Jour, Semaine ou Mois.
  • Vérifiez pour modifier automatiquement la cote de l'axe des x.

Les données

Les données suivantes sont affichées dans le graphique:

  • Essais en cours: nombre de clients participant à des essais par abonnement
  • New Trials - Nombre de nouveaux abonnements d'essai
  • Gain: nombre de nouveaux clients
  • Net - Le nombre de nouveaux clients a augmenté moins le nombre de clients perdus
  • Lost: nombre de clients ayant annulé un abonnement d'essai ou un abonnement payant
  • Lost Trials: nombre d'essais d'abonnement annulés

Jalons

Créez un jalon pour suivre son impact sur le nombre de clients.

Actions

Vous pouvez sélectionner l'une des actions suivantes:

  • Cliquez sur Afficher pour filtrer les données par:
    • Opérations
    • Produit
    • Affaires
    • Contenus
    • Marketing

  • >Cliquez pour créer un nouveau jalon. Remplissez les champs du formulaire et sélectionnez AJOUTER lorsque vous avez terminé.


  • Survolez le graphique pour voir les détails des données ainsi que le jalon correspondant.
