Clients par catégorie

Afficher le plus grand nombre de spectateurs par catégorie. Les 10 premiers sont affichés dans le graphique, avec une liste détaillée à droite du graphique.

Actions

Vous pouvez sélectionner l'une des actions suivantes pour modifier les données présentées:

  • Cliquez pour sélectionner la catégorie des données affichées.

  • Survolez pour voir la remarque suivante:
    • Ces informations sont un aperçu actuel de votre entreprise.

Les données

Les données suivantes sont affichées dans le graphique:

  • Clients (#) - L'axe des ordonnées indique le nombre de clients pour chaque catégorie ; chaque barre du graphique montre également les revenus de ces clients
  • Basic (Monthly) - L'axe des x montre les types de données pour la catégorie choisie ; dans ce cas, c'est par plan
  • Plan - Le plan d'abonnement avec la hausse ou la baisse du nombre de clients
    • LTV - Lifetime Value (LTV) estime le montant des revenus qu'un client représente pour une entreprise au cours de la durée de vie de la relation ; cela est utile pour déterminer la rentabilité de l'acquisition de nouveaux clients et de leur soutien au fil du temps
    • MAU - Utilisateurs actifs mensuels (MAU) ; les utilisateurs actifs sont définis comme étant connectés
  • Channel: l'App Store à partir duquel les utilisateurs ont consulté votre chaîne
  • Appareil: appareil à partir duquel les utilisateurs ont consulté votre contenu
