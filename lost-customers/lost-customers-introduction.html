Présentation

La section Clients perdus vous permet de suivre le nombre de clients perdus au fil du temps et vous fournit des informations sur les clients qui doivent annuler des abonnements. Cela vous permet de contacter de manière proactive ces clients potentiellement perdus.

Dans votre tableau de bord Audience Insights, sélectionnez l'onglet Clients perdus pour voir les informations.

Clients perdus dans le passé

Dans la section Clients perdus, cliquez sur le bouton du mois pour sélectionner la date spécifique pour vos données de rapport.
