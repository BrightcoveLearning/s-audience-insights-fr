Consultez les impressions vidéo, les installations, les activations et les premiers paiements pour chaque boutique d'applications.

Actions

  • Cliquez pour télécharger les données d'abonnement sous forme de fichier CSV ou d'image PNG.

Les données

Survolez le graphique pour afficher les données.

  • Impressions: nombre de fois qu'une vidéo a été chargée dans un lecteur et prête à être utilisée
  • Installs: nombre de fois où les utilisateurs ont installé votre application depuis une boutique
  • Activations: nombre de fois où les utilisateurs ont activé votre application
  • Premiers paiements: nombre de premiers paiements d'abonnement au cours de la période sélectionnée
