Abonnements tiers

Consultez les données des abonnés sur des plateformes tierces, notamment le Roku Store, le Google Store, l'Apple Store et Amazon Prime Channel.

Actions

Voici les actions disponibles:

  • Cliquez pour filtrer les données des abonnés en:

    • Tout
    • Applications X1
    • Boutique Roku
    • Google Store
    • Apple Store

    • Chaîne Amazon Prime

  • Cliquez pour télécharger les données d'abonnement sous forme de fichier CSV ou d'image PNG.

  • Sélectionnez une plage de dates pour vos données. Notez que la plage de dates doit être de 5 jours ou plus.

Les données

Survolez le graphique pour afficher les données, y compris les jalons:

  • Nouveaux essais: nombre de nouveaux abonnements d'essai sur une plateforme tierce
  • Nouveaux abonnés: nombre de nouveaux abonnements sur une plateforme tierce
  • Abonnés nets - Nouveaux abonnements moins annulations
  • Annulations: nombre de clients ayant annulé leur abonnement sur une plateforme tierce

Jalons

Créez un jalon pour suivre l'impact que cela aura sur vos abonnés convertis.

Actions

Vous pouvez sélectionner l'une des actions suivantes:

  • Cliquez sur Afficher pour filtrer les données par:
    • Opérations
    • Produit
    • Affaires
    • Contenus

    • Marketing

  • >Cliquez pour créer un nouveau jalon. Remplissez les champs du formulaire et sélectionnez AJOUTER lorsque vous avez terminé.


  • Survolez une entrée de jalon pour voir sa position dans le graphique. Survolez le graphique pour voir les données correspondantes.
