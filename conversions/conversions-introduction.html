Présentation

Audience Insight Data intègre vos sources de données vidéo OTT (Over-the-Top) à des informations sur les vidéos d'abonnement pour fournir des informations exploitables basées sur des benchmarks clés du secteur.

Dans votre tableau de bord Audience Insight, sélectionnez l'onglet Conversions pour voir les données relatives à vos abonnés convertis.

Cliquez pour sélectionner une période pour vos données de conversion.
