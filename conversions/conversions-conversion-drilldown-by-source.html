Expliner les conversions par source

Consultez des détails supplémentaires pour chaque source de référence pour connaître la taille et l'efficacité relatives.

Actions

Les actions suivantes sont disponibles:

  • Cliquez pour sélectionner la source de référence pour les données affichées.

  • Cliquez pour télécharger les données de conversion sous forme de fichier CSV.

  • Cliquez sur une étape pour trier la colonne par ordre décroissant.
