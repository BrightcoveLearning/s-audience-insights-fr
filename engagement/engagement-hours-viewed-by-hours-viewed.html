Nombre d'heures visionnées par catégorie

Les statistiques des heures consultées vous permettent de voir les heures visionnées par forfait, canal, appareil et lieu de visionnage.

Dans la section Heures consultées par, cliquez sur le bouton du plan pour sélectionner une catégorie pour vos données de reporting.

Lorsque vous cliquez sur les options suivantes, différents graphiques s'affichent.

  • Plan

  • Chaîne

  • Appareil

  • Lieu de visionnement

Actions

Voici les actions disponibles pour ce graphique.

  • Cliquez pour télécharger les données sous forme d'image PNG ou PDF.

  • Cliquez pour effectuer un zoom avant ou arrière sur le graphique.
